About

ulti application

This application was created to help users provide health information related to diabetes based on measurement values such as Pregnancies, Glucose, Blood Pressure, Skin Thickness, Insulin, BMI, Diabetes Pedigree Function, Age.

The user only needs to provide a measurement value to be able to find out the condition experienced, which includes diabetes or not.
